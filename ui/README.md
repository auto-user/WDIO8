### Requirements:
NodeJS 16 or above
Vscode

### Getting Started
Clone Repository
```bash
1. git clone 
2. Navigate to 'ui'
```

For UI E2E test:
Install the dependencies from the terminal:
```bash
npm install
```

Run tests:
```bash
npm test (to run all tests)
```

Generate allure report:
```bash
npm run report
```

Emailable HTML Report:
```
<ROOT_PROJECT_FOLDER>/reports/html/index.html
```
